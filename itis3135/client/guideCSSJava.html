CSS and Javascript Guide

Try out this guide on the creative tab -> Creative

CSS is how we style our HTML and is stored in a style sheet or done with embedded sytle tags.
This guide will explain embedded style sheet purely to use the creative tab,
to change to an embedded a link tag must be used in the head tag such as this:
<link rel="stylesheet" type="text/css" href="styles/design.css"<
By using the conent of the style tags shown in a .css file, the external method may be used.

<body>- At the top of the body (the top of the text box) place a style tag.
Step 1: <style>- Creates a style tag opening in the HTML.

Step 2: #p1 {
color: red;
}
This creates an 'id' for a part of the HTML, Ids are one use only, followed by the style
changes, such as the text color being red. Lastly the close ends the style changes for id p1.

Step 3: <p id="p1">Hippos are dangerous </p>
This creates a p tag with the id 'p1' to use the style changes in the style tags!

Website Structure

This website consists of multiple pages, each dedicated to a specific portion of designing a tutorial website, to follow along use the pages in this order:

HTML Tutorial - Explains the fundimentals of HTML and how to set up the core of a website, as well as gives a basic introduction into applying HTML for a tutorial website.

CSS/Javascript Tutorial - Explains both the fundimentals of CSS (The design language) and Javascript (The functions language), as well as gives a basic introduction into applying both to the tutorial website.

Creative Page - Contains both a Design and Creative page- by writing HTML/CSS/Javascript in the textbox on the design page, the submit button can be pressed to apply the written code to the creative page to view what website would look like.

Contact Page - Contains all contact emails for any issues on each page of the site.

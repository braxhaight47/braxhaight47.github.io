HTML is comprised of 'tags' which look like this <> and add parts to the website.
For every opening tag there is a closing tag that looks like this </>
By combining these tags and putting in content, you can create a webpage!
The following is the set up of a page, and is preset for the creative page so there is no need to type it.

<!DOCTYPE html>- Sets the type of the page to HTML as it is a HTML file.
<html lang="en">- Sets the default language to english.
<head>- This is the head tag, used for setting the title and linking style sheets.
<title>- This creates the title of the page, or what is written in the websites tab.

</title>- This is a closing tag for title
</head>- This is a closing tag for head
<body>- This is a body tag, all the main parts of the website are placed here.

This is all the preloaded code that establishes the body of the HTML for this and the creative page.

Step 1: <h2>This is a header!</h2>
The 2 in h2 specifies the size, they go from 1-7 with 1 being the heading of the page.

Step 2: <p>This is a text block!</p>
The text block allows chunks of text to be added such as this very sentence!

Step 3: <a href="">This is an anchor tag! It creates a link to any website in the quotes.</a>
The anchor tag will allow you to link other websites or pages to the anchor tag.

This is the closing of the preset tags in the beggining of the HTML from the beggining of the website.
</body>- This is the closing body tag.
</html>- This is the closing of the html tag and end of the website!

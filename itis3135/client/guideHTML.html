HTML Tutorial

Section 1: Introduction to HTML

HTML, or HyperText Markup Language, is the standard language for creating web pages and web applications.

It consists of a set of tags and attributes that define the structure and content of a web page, such as headings, paragraphs, links, images, and more.

By combining these tags the structure of a website is created and can be filled with content.

THe HTML is only the first step as stated before, HTML is simply the structure and content- not the design portion.

The design of a website stems from that sites CSS which is either internal (Written in the html) or external (Written in an outside file that needs to be linked).

Besides design, most websites typically have other functions that they can preform that are more complicated than HTML can do- which is where Javascript comes in.

Just like CSS, Javascript can be internal or external, but instead of design- Javascript allows many things to be done such as how this portion of the guide is set up to expand each section individually.

Putting all of these peices together creates an ideal website! The following sections of this guide will detail how to construct the HTML portion.

Section 2: Basic HTML tags

Here are some of the most common HTML tags:

  • <p> - defines a paragraph of text
  • <a> - defines a hyperlink to another web page or location
  • <img> - defines an image on the page
  • <h1> to <h6> - define headings of different sizes
  • <div> - defines a division or section of the page
  • <span> - defines a span of inline text

Section 3: Basic Structure

All websites have to have a simlar broad structure in order to have HTML understand how to format the website properly.

That format is a template for website design and is as follows:

<!DOCTYPE html> - Defines that this whole page will be in html.
<html lang="en"> - Defines that this website will be in English and starts the portion of code that is html.
<head> - Defines the head of the website, which is where all external links (CSS sheets or Scipts- explained more in the CSS/JS Tutorial Page) are placed as well as the title.
<title> Title of Website </title> - Defines what the title of the website will be (The portion that is in the tab on the browser).
<link rel="stylesheet" type="text/css" href="name_of_style_sheet.css"> - This is where the link to an external style sheet will go.
<script src="name_of_script.js"><script> - This is where the link to an external script will go.
</head> - Closes the head tag.
<body> - Defines the body, or content portion, of the website.
<header> - Defines the header, or the top/navigation portion, of the website.
<h1>Heading of Website</h1> - Defines the heading of the website, such as this websites is 'How to Create a Hippo Themed Website'.
<nav> - Defines the navigation bar of a website if there are multiple pages.
</nav> - Closes the navigation tag.
</header> - Closes the header tag.
<main> - Defines the start of all the content of the website such as this very line.
</main> - Closes the main tag.
<footer> - Defines the footer, or the portion that is at the bottom of the website, such at the area that has the link to Haight Enterprise on this page.
</footer> - Closes the footer tag.
</body> - Closes the body tag.
</html> - Closes the initial html tag.
